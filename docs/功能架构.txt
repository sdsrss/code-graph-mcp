  数据流：
  1. 索引流：文件变更 → Merkle diff 检测 → Tree-sitter 解析为 AST 节点 → 提取函数/类/路由/调用关系 → 图上下文注入 → candle 生成嵌入 → 写入 SQLite
  2. 查询流：MCP 工具调用 → 按需增量检查 → FTS5/Vec0/Graph 三引擎并行查询 → RRF 融合排序 → Context Sandbox 压缩 → 返回结果

  模块划分（src/ 下）：
  src/
  ├── main.rs              // 入口，初始化
  ├── mcp/                 // MCP 协议层
  │   ├── mod.rs

  │   ├── mod.rs
  │   ├── store.rs         // edges 表 CRUD
  │   └── query.rs         // 递归CTE查询（调用链/继承链）
  ├── embedding/           // 嵌入引擎
  │   ├── mod.rs
